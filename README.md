# Análisis de Recursos Humanos - Empresa SRL

## Contexto y Objetivo
Se analizó una base de datos con información detallada sobre la contratación, desempeño y características demográficas de los empleados de la compañía SRL. El objetivo de este proyecto es proveer al departamento de Recursos Humanos con un dashboard interactivo que permita identificar puntos clave para optimizar la gestión del talento.

## Resultados del Análisis
Basado en los gráficos creados en Tableau, se obtuvieron los siguientes insights estratégicos:

1. **Salario por Departamento:**
Se analizó el salario promedio por cada área de la empresa. Esto nos permite observar cómo se distribuye el presupuesto de nómina y evaluar si existe equidad salarial interna, lo cual es fundamental para mantener la motivación del personal.

2. **Mejores Fuentes de Reclutamiento:** 
Al contabilizar el número de empleados atraídos por cada canal, identificamos visualmente que plataformas como *Indeed* y *LinkedIn* son las fuentes más efectivas. Esto le indica a Recursos Humanos que deben concentrar su presupuesto y esfuerzos de atracción en estas redes.

3. **Razones de Terminación de Contrato:** 
Se filtró y contabilizó al personal inactivo para descubrir los motivos principales por los que abandonan la empresa (como encontrar otra posición, insatisfacción o cambio de carrera). Conocer estos focos rojos es el primer paso para diseñar estrategias de retención y mejorar el entorno laboral.
